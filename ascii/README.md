#  Лаба 1 (усложненная)
Написать программу-загрузчик, выводящую на экран ascii-таблицу.
## Запуск 
```shell
nasm -fbin ascii.asm -o ascii.bin && qemu-system-i386 -drive file=ascii.bin,format=raw
```
